# Lucky

> 纯前端的抽奖程序
> 
> 依赖客户端 localstorage，所以设置和抽取不能更换浏览器。


### 基于组件

- vite
- vue v3+
- vue-router v4+
- vue-i18n v9+
- eslint airbnb
- prettier v2+
- lodash v4+
- antdv v3+

### 安装说明

```
npm install
```

### 配置

**管理入口**

```angular2html
/login
```

**账号密码设置**

在 .env 里设置

```angular2html
账号：admin
密码： admin123
```

**标题设置**

在 .env 里设置 subject
